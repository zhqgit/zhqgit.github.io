windows系统常用快捷键

  1. 【Windows键 + D】高手们常用的键，瞬间回到桌面。

  2. 【Windows键 + E】直接打开“我的电脑”。

  3. 【Windows键 + L】电脑锁屏，不用担心隐私外泄。

  4. 【Windows键 + R】再输入osk，点“运行”，立刻弹出虚拟键盘。

  5. 【Ctrl + Tab】快速切换浏览器选项卡。

  6. 【Alt + F4】快速关闭当前程序；在桌面用则调出关机选项。

  7. 【Ctrl + F】快速查找电脑里的文件，类似于网络中的搜索引擎。

  8. 玩游戏时突然有人来，按【Alt + Tab】，立马切换窗口。

  9. 浏览网页或图片时，按【F11】即实现全屏清晰浏览。

  10. 【Ctrl + C】复制；【Ctrl + V】 粘贴。
